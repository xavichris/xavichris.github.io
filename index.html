What does this code do?

It demonstrates the usage of subtitles in <video> tag. All files used are hosted in Dropbox, from .mp4 file, to .png thumbnail, and .vtt subtitle tracks. There is NO crossorigin attribute used.

To change subtitles, click on the ⋮ part of the video player for options, select CC Captions to change language of subtitles.

The default language is English [EN]. You can change the language of subtitles to:
  • Japanese [JA] - 日本語
  • German [DE] - Deutsch
  • Spanish [ES] - Español
  • Czechia [CZ] - Česko
  • French [FR] - Français
  • Italian [IT] - Italiano
  • Russian [RU] - Русский
  • Hindi [HI] - हिंदी
